SocialConnectedness documentation
=================================

This is a library which provides command line scripts for processing data collected by the `Beiwe smartphone app`_.
The main features can be seen in the `Command Line Interface`_ section.
Documentation for internal functions, which may be useful to future developers, can be seen in the API section.
